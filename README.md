# jpa-crud-api
A simple RESTful API for managing **Dog** objects.  
Built with Spring Boot, JPA (Hibernate), and PostgreSQL.

## Installation

### 1. Get the project
```
Clone the repository: https://github.com/Meylin30/jpa-crud-api
```
- OR download zip.
- Open the project in VS Code.
- This project is built to run with jdk 21.
- [`Dependencies`]([[[https://github.com/uncg-csc340/su25-jpa-crud-api/blob/3149ec363e4aae4baebe6f755df7d4c2d79c9d2c/pom.xml#L32](https://github.com/Meylin30/jpa-crud-api/blob/main/pom.xml)](https://github.com/csc340-uncg/f25-jpa-crud-api/blob/6b2860c4ad01ca46b6b62852ca966bfadc8dfc6a/pom.xml#L32](https://github.com/Meylin30/jpa-crud-api/blob/main/pom.xml))) to JPA and Postgres in addition to the usual Spring Web. JPA handles the persistence, Postgresql is the database to be used.
- [`/src/main/resources/application.properties`]([https://github.com/csc340-uncg/f25-jpa-crud-api/blob/6b2860c4ad01ca46b6b62852ca966bfadc8dfc6a/src/main/resources/application.properties](https://github.com/Meylin30/jpa-crud-api/blob/main/src/main/resources/application.properties)) This file has the configuration for the PostgreSQL database to use for the API.
  - You MUST have the database up and running before running the project!
    - Login to your neon.tech account.
    - Locate your database project.
    - On the project dashboard, click on "Connect" and select Java.
    - Copy the connection string provided.
    - Paste it as a value for the property `spring.datasource.url`. No quotation marks.
- Build and run the main class. You should see a new table created in the Neon database.

- ## API Endpoints
Base URL: [`http://localhost:8080/Dogs`](http://localhost:8080/dogs)


1. ### [`/`](http://localhost:8080/dogs) (GET)
Gets a list of all Dogs in the database.

#### Response - A JSON array of Dog objects.

 ```
[
  {
    "Id": 1,
    "name": "Luna",
    "breed": "Pomeranian",
    "age": 1,
    "description": "A Pomeranian is a small, fluffy dog with a fox-like face, perky ears, and a lively, affectionate personality."
    }

  {
    "id": 2,
    "name": "Max",
    "breed": "German Shepherd",
    "age": 2,
    "description": "A German Shepherd is a large, intelligent, and loyal dog breed known for its strength, versatility, and protective nature."
  }
]
```

2. ### [`/{Id}`](http://localhost:8080/dogs/2) (GET)
Gets an individual dog in the system. Each dog is identified by a numeric `Id`

#### Parameters
- Path Variable: `Id` &lt;integer &gt; - REQUIRED

#### Response - A single Student

```
  {
    "id": 2,
    "name": "Max",
    "breed": "German Shepherd",
    "age": 2,
    "description": "A German Shepherd is a large, intelligent, and loyal dog breed known for its strength, versatility, and protective nature."
  }
```

3. ### [`/name`](http://localhost:8080/students/search?name=Mat) (GET)
Gets a list of dogs with a name that contains the given string.

#### Parameters
- query parameter: `search` &lt; String &gt; - REQUIRED

#### Response - A JSON array of Dog objects.

```
[
  {
     "id": 3,
    "name": "Mat",
    "breed": "Siberian Husky",
    "age": 1,
    "description": "A Siberian Husky is a medium-sized, energetic dog with a thick double coat, striking eyes, and a friendly, adventurous temperament."
  }
]
```

4. ### [`/category/{category}`](http://localhost:8080/dogs/breed/Pomeranian) (GET)
Gets a list of breeds for each dog.

#### Parameters
- path variable: `breed` &lt; String &gt; - REQUIRED

#### Response - A JSON array of Dog objects.

```
[
  {
   "id": 1,
    "name": "Luna",
    "breed": "Pomeranian",
    "age": 1,
    "description": "A Pomeranian is a small, fluffy dog with a fox-like face, perky ears, and a lively, affectionate personality."
  }
]
```

5. ### [`/`](http://localhost:8080/dogs) (POST)
Create  a new dog entry

#### Request Body
A dog object. Note the object does not include an ID as this is autogenerated.
```
{
  "name": "Jack",
  "breed": "Golden Retriever",
  "age": 3,
  "description": "A Golden Retriever is a friendly, intelligent, and gentle dog with a dense golden coat and a loving, loyal temperament."
}
```
#### Response - The newly created Dog.

```
  {
   "id": 7,
  "name": "Jack",
  "breed": "Golden Retriever",
  "age": 3,
  "description": "A Golden Retriever is a friendly, intelligent, and gentle dog with a dense golden coat and a loving, loyal temperament."
  }
```

7. ### [`/{Id}`](http://localhost:8080/Dogs/3) (PUT)
Update an existing Dog.

#### Parameters
- Path Variable: `id` &lt;integer&gt; - REQUIRED

#### Request Body
A dog object with the updates.
```
{
  "id": 3,
  "name": "Mat",
  "breed": "Siberian Husky",
  "age": 1,
  "description": "A Siberian Husky is a medium-sized, energetic dog with a thick double coat, striking eyes, and a friendly, adventurous temperament."
}
```
#### Response - the updated dog object.
```
{
  "id": 3,
  "name": "Mat",
  "breed": "Siberian Husky",
  "age": 2,
  "description": "A Siberian Husky is a medium-sized, energetic dog with a thick double coat, striking eyes, and a friendly, adventurous temperament."
}
```
